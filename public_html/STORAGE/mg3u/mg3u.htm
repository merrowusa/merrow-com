Merrow Optical Auto-Emblem

The Merrow Optical Auto-Emblem Machine (MG3U AEM) is a total automation solution. Once loaded with blank emblems, the MG3U AEM can literally run unattended until the machine needs to be reloaded. And, even reloading can be done without stopping the machine.

The efficiencies gained by the MG3U AEM are so dramatic that one operator can operate up to six sewing heads at a time (four is typical), running at speeds that can reach 4000 s.p.m. The net result: a high return on investment that begins the moment the machine is installed.
