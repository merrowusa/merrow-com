Crochet Shell Applications

  
 

  Blankets, fleece garments, lightweight sweaters, infant's wear, ladies' golf socks, curtains, swimwear.

The following crochet shell machine variants are also available:
22-, 27-, 35-FJT   The “T” model permits tape to be included within the shell to act as a draw string.

22-, 27-, 35-FJT-3   The “FJT-3” version produces three different stitches on the same machine (plain crochet, plus 4- and 8-stitch shells), which can be selected by simply shifting a lever.
